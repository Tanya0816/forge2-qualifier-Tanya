# Kanban Builder Skill

Purpose:
Coordinate the creation of a Trello-style Kanban board.

When asked to build or modify the Kanban board:

1. Analyze the request.
2. Create an implementation plan.
3. Break work into tasks.
4. Assign coding tasks to OpenClaw.
5. Monitor progress.
6. Report status using:

What I Did:
What's Left:
What Needs Your Call:
